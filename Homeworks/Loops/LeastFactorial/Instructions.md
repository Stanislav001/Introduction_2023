
# Least Factorial

### Instructions


Given an integer  `number`, find the  _minimal_  `k`  such that

-   `k = m!`  (where  `m! = 1 * 2 * ... * m`) for some integer  `m`;
-   `k >= number`.

In other words, find the smallest factorial which is not less than  `number`.
  

**Examples:**

*Input* : 17

*Output* :  24
